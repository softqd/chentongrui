相对定位与绝对定位的区别

  • 相对定位 (position: relative)：元素相对于其正常位置进行偏移，但保留原始空间
  • 绝对定位 (position: absolute)：元素脱离文档流，相对于最近的非static定位祖先元素定位
  • 相对定位常用于微调元素位置，不影响其他元素的布局
  • 绝对定位常用于创建悬浮元素、弹出层等需要脱离文档流的场景
  • 示例中相对定位的元素保留了原始空间，而绝对定位的元素不占用布局空间

CSS8 - 核心概念展示

CSS 核心概念：背景、盒模型、布局、动画

背景

"cascading" 的音标是 /kæˈskeɪdɪŋ/。CSS背景属性允许您设置元素的背景效果， 包括背景颜色、背景图片、背景定位和重复方式等。

CSS背景属性包括：

  • background-color: 设置背景颜色
  • background-image: 设置背景图片
  • background-position: 设置背景位置
  • background-repeat: 设置背景重复方式
  • background-size: 设置背景尺寸

盒模型

盒模型是 CSS 布局的基石，它定义了元素在页面上的位置和大小。在 CSS 中，每个 元素都被看作是一个矩形的盒子，这个盒子由内容、内边距、边框和外边距组成。

box 中的内容 2

CSS盒模型属性包括：

  • width 和 height: 内容区域的宽度和高度
  • padding: 内容与边框之间的内边距
  • border: 元素的边框
  • margin: 元素与其他元素之间的外边距

动画

CSS动画允许您创建平滑的过渡和动画效果，而不需要使用JavaScript。CSS动画由两个主要部分组成：

  • @keyframes: 定义动画序列
  • animation 属性: 将关键帧应用到元素

动画必须要有起始状态、结束状态和持续时间。
