JavaScript 核心语法学习

掌握变量、数据类型、运算符、控制结构和函数等基础知识

变量与数据类型

// 变量声明 let a = 100; const api_key = 'ssssx-=-dddd'; // 类型检测 console.log(typeof a); // "number" console.log(typeof api_key); // "string" console.log(typeof true); // "boolean" // 类型转换 let b = '1'; console.log(a + b); // "1001" (字符串拼接) console.log(a + Number(b)); // 101 (数值相加) // 相等性比较 let c = '100'; console.log(a == c); // true (值相等) console.log(a === c); // false (类型不同)

let 用于声明可重新赋值的变量，const 用于声明常量（不可重新赋值）。

typeof 操作符返回一个字符串，表示未经计算的操作数的类型。

JavaScript 是弱类型语言，会进行隐式类型转换（如数值和字符串相加）。

使用 === 严格相等运算符可以同时比较值和类型。

控制结构

// 分支语句 let current_time = '12:20'; if (current_time == '12:20') { console.log('冲啊！吃午饭时间到了！'); } else { console.log('假装学习……'); } // for循环 for (let i = 1; i <= 5; i++) { console.log('for循环计数: ' + i); } // while循环 let hug = 60; while (hug > 0) { console.log('继续干饭... 饥饿值: ' + hug); hug -= 10; }

if/else 语句根据条件执行不同的代码块。

for 循环在已知循环次数时使用，包含初始化、条件判断和迭代器。

while 循环在满足条件时重复执行代码块，适用于不确定循环次数的情况。

函数基础

// 无参数函数 function intro() { console.log('你好，我是一个脚本！'); } intro(); // 带参数函数 function intro2(content) { console.log('以下是我介绍的内容: ' + String(content)); } intro2('这是传入的参数'); // 多参数函数 function intro3(content1, content2) { console.log('多参数内容: ' + String(content1) + String(content2)); } intro3('参数1', '参数2', 444); // 多余参数会被忽略 // 带返回值的函数 function intro4(content) { console.log(content); return content + '这是返回值'; } let result = intro4('带返回值的函数'); console.log(result);

函数是执行特定任务的可重用代码块。

函数可以接受参数（输入）并返回值（输出）。

使用 function 关键字定义函数，通过函数名加括号调用。

return 语句指定函数的返回值（没有return时返回undefined）。

JavaScript 学习要点

变量声明

使用 let 和 const 代替过时的 var。const 用于声明常量，let 用于声明可重新赋值的变量。

相等性比较

优先使用严格相等 === 和严格不等 !==，避免类型转换导致的意外结果。

函数设计

遵循单一职责原则，每个函数只做一件事。合理使用参数和返回值，避免函数副作用。

代码风格

使用一致的缩进（通常2或4个空格），在操作符周围添加空格，使用驼峰命名法，添加有意义的注释。

调试技巧

善用 console.log() 输出调试信息，使用浏览器开发者工具设置断点和检查变量值。

JavaScript 是动态类型语言，变量类型在运行时确定。

JavaScript 使用原型继承而非类继承（ES6类语法是语法糖）。

理解作用域和闭包是掌握JavaScript的关键。
